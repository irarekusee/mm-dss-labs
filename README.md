# X-матрица
**Лабораторная работа №0**
**Тема:** план и результативность прохождения учебного курса в соответствии с ожиданиями.
**Цель работы:** научиться работать с Х-матрицей - формой представления активностей в целевом проектировании.
**Задачи:** заполнить Х-матрицу в соответствии со следующей структурой.
- Север - ожидания, проблемы, идеи;
- Запад - научно-технологические методы, средства;
- Восток - компетенции как результат обучения;
- Юг - список учебных мероприятий или тем;
- Угловые субматрицы - сила связей между строками (Х - на уровне понимания, ХХ - применение, ХХХ - создание нового).
**Рефлективное заключение:** в ходе выполнения лабораторной работы был получен навык работы с Х-матрицей, позволяющей составлять стратегию для достижения желаемых результатов. Стоит отметить, что такой инструмент позволяет системно смотреть на процесс профессионального развития. Так, на мой взгляд, выстраивание чёткой схемы мероприятий с учётом собственных ожиданий может поспособствовать более эффективному преобретению знаний предметной области.
